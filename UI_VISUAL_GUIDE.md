## Interaction Flow

### Text Summarization Flow
```
1. User selects language (e.g., "Oromiffa")
2. User types text in input field
3. User presses Enter or clicks Send button
4. Loading indicator appears
5. Summary appears in selected language
6. Audio player becomes available
```

### File Summarization Flow
```
1. User selects language (e.g., "Amharic")
2. User uploads file (drag & drop or click)
3. File message appears in chat
4. Loading indicator appears
5. Summary appears in selected language
6. Audio player becomes available
```
